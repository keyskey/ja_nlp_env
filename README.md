# ja_nlp_env

My special environment for Japanese text's NLP with pre-processing unit

## Setup

```
git clone git@github.com:keyskey/ja_nlp_env.git
cd ja_nlp_env
make build
make run
```

Access to the url shown in your console and open jupyterlab
